# PyMin – 日経225mini 1分足 OHLC自動生成ツール

**PyMin** は、カブステーション API を利用して  
日経225mini のリアルタイム価格から1分足の OHLC（Open, High, Low, Close）を自動生成・保存する Python アプリケーションです。

---

## 📌 主な特徴

- kabuステーションの WebSocket 配信から現値を取得
- ティックデータから 1分足 OHLC をリアルタイムで構築
- 日中・夜間セッションの判定、自動限月切替に対応
- 取引日ごとに CSV ファイルへ自動保存
- ログ出力（タイムスタンプ付き）を自動記録
- 設定は `settings.json` から柔軟に管理可能
